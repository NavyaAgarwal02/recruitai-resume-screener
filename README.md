## What Is This?

**RecruitAI** is a fully functional, AI-powered resume screening web application that helps HR teams evaluate multiple candidates against a Job Description — in under 10 seconds.

No spreadsheets. No manual reading. No unconscious bias.
Just paste a JD + resumes → get ranked candidates with scores, strengths, gaps, and hiring recommendations — instantly.

## Features

| Feature | Description |
|---|---|
| 🤖 Claude AI Integration | Uses Anthropic's claude-sonnet-4 model via REST API |
| 📊 Match Scoring (0–100) | Every candidate scored objectively against the JD |
| 🏆 Auto Ranking | Candidates sorted by score — best fit appears first |
| 💪 Strengths Analysis | 2–3 key strengths extracted per candidate |
| ⚠️ Gap Identification | 2–3 skill/experience gaps flagged per candidate |
| 🎯 Fit Recommendation | Strong Fit / Moderate Fit / Not Fit label |
| 📋 Summary Dashboard | Total screened and fit counts at a glance |
| ⚡ Real-Time Processing | 5 resumes screened in under 10 seconds |
| 📱 Fully Responsive | Works on desktop, tablet, and mobile |

## How It Works

```
┌──────────────────┐       ┌─────────────────────┐       ┌──────────────────────┐
│  Job Description │       │                     │       │  Ranked Candidates   │
│  + Resumes 1–10  │──────▶│   Claude AI API     │──────▶│  Score, Strengths,   │
│                  │       │   (claude-sonnet-4) │       │  Gaps, Fit Label     │
└──────────────────┘       └─────────────────────┘       └──────────────────────┘
```

**Step-by-step:**

1. User pastes a **Job Description** and **1–10 resumes** (separated by `---RESUME BREAK---`)
2. App sends a carefully engineered prompt to the **Claude API**
3. Claude evaluates each resume and returns clean **JSON output**
4. App parses and renders results as a **ranked interactive dashboard**
5. HR clicks any candidate card to see full strengths, gaps, and hiring rationale

## Run Locally

```bash

npm install

# 4. Start dev server
npm start

```

> via Anthropic's /v1/messages endpoint.
